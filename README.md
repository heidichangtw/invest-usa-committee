# invest-usa-committee Website

A concept website for the Invest USA Committee, designed to provide information and resources for Taiwanese companies exploring investment opportunities in the United States.

## Website Sections

- Home
- About Us
- Members
- Invest
  - Why Invest in the U.S.
  - How to Invest in the U.S.
  - Attract Taiwanese Investment
- Event
- Resources
- Contact

## Features

- Interactive Taiwan FDI map
- Information on the Invest USA Committee
- Membership information
- U.S. investment resources for Taiwan companies

## Technologies

- HTML
- CSS
- JavaScript
